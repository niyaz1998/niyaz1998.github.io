# Введение
репозиторий для статических файлов rosbank.ru проекта инвест

## pwa
в папке находятся сборки PWA приложения Росбанк Инвест, именование папок происходит по дефолтному серверу, к которому обращается приложение.

## deeplink redirect
Страница перенаправляющая на диплинк приложения rosbank invest. 
В данный момент мы перенавпрялем на firebase dynamic link, который просто октрывает приложение или сторы.
В будущем планируется изменить ссылку, так как планиурется переход на branck.io.
